환경 조건 해석 가이드

1. 기상 조건
   - 맑음: 정찰, 기동, 공중 지원 유리
   - 비: 기동 제약, 통신 간섭 가능
   - 안개: 정찰 제약, 기습 효과 증가
   - 강풍: 공중 작전 제약, 화력 정확도 저하

2. 시간 조건
   - 주간: 정찰 유리, 가시거리 확보
   - 야간: 기습 유리, 정찰 제약
   - 황혼/새벽: 전환 시간, 경계 태세 중요

3. 지형 조건
   - 평지: 기동 유리, 방어 어려움
   - 산지: 방어 유리, 기동 제약
   - 구릉지: 기동과 방어 균형
   - 개활지: 화력전 유리, 은엄 어려움
   - 도시지역: 근접전, 민간인 고려 필요

4. 가시거리
   - 5km 이상: 장거리 화력전 유리
   - 2-5km: 중거리 교전
   - 2km 미만: 근접전, 정찰 중요

5. 환경 조건별 방책 적합성
   - 맑음 + 평지: 공격/기동 방책 유리
   - 비 + 산지: 방어 방책 유리
   - 안개 + 야간: 기습 방책 유리
   - 강풍: 지상 작전 우선, 공중 작전 제약

6. 환경 조건 해석 시 주의사항
   - 복합 조건 고려 (기상 + 지형 + 시간)
   - 조건 변화 예측 (기상 예보)
   - 대체 방책 준비 (조건 변화 시)
   - 자원 적응성 확인 (다양한 조건 대응 가능 여부)
