COA 적용조건 해석 가이드

1. 고위협 상황 (threat_level > 0.8)
   - 적의 공격 의지가 높고 전투력이 우세한 경우
   - 방어 우선 방책 검토
   - 자원 집중 배치 필요

2. 저위협 상황 (threat_level < 0.3)
   - 적의 공격 의지가 낮고 전투력이 열세인 경우
   - 공격 또는 기동 방책 검토 가능
   - 자원 효율적 배치 가능

3. 침투 상황 (penetration == True)
   - 적이 아군 방어선을 돌파한 경우
   - 즉각 대응 방책 필요
   - 역습 또는 차단 방책 검토

4. 자원부족 상황 (resources < 0.5)
   - 가용 자원이 제한적인 경우
   - 기동 방책 우선 검토
   - 자원 집약적 방책 회피

5. 적기세약화 (enemy_momentum < 0.5)
   - 적의 공격 기세가 약화된 경우
   - 반격 또는 공격 방책 검토
   - 전환점 활용 필요

6. 보급차단 (logistics_cut > 0.7)
   - 적의 보급선이 차단된 경우
   - 지연전 또는 소모전 방책 검토
   - 시간 활용 전략

7. 측면노출 (flank_exposed == True)
   - 적의 측면이 노출된 경우
   - 기동 방책으로 우회 공격 검토
   - 기습 효과 극대화

8. 화력우위 (firepower > 0.8)
   - 아군 화력이 우세한 경우
   - 정면 공격 방책 검토
   - 화력 집중 전략

9. 예비대가용 (reserve_available == True)
   - 예비대가 가용한 경우
   - 반격 또는 기동 방책 검토
   - 전술적 유연성 확보
