## 1. 시작하기

### 1.1 접속 방법
웹 브라우저(Chrome 권장)를 실행하고 다음 주소로 접속합니다.
- **URL**: `http://localhost:5173`

### 1.2 메인 화면 구성
시스템은 직관적인 **C2 Dashboard** 인터페이스를 제공합니다.
- **Top Navigation Bar (GNB)**:
    - **COMMAND & CONTROL**: 메인 관제 및 작전 수행 화면
    - **ONTOLOGY STUDIO**: 지식 그래프 탐색 및 데이터 확인
    - **DATA MANAGEMENT**: 원천 데이터 관리
    - **SYSTEM GUIDE**: 도움말 및 기술 문서
- **Dashboard Layout**:
    - **Left Panel**: 상황 입력 및 시나리오 선택
    - **Center Panel**: 방책 추천 결과 및 분석
    - **Right/Bottom Panel**: 세부 정보, 채팅, 로그

## 2. 주요 기능 및 워크플로우

시스템의 핵심 사용 흐름은 다음과 같습니다.

1.  **상황 식별**: 적군 활동이나 위협 징후를 입력합니다.
2.  **분석 요청**: Agent에게 방책 추천을 요청합니다.
3.  **결과 확인**: 추천된 방책(COA)의 순위와 근거를 확인합니다.
4.  **시각화 분석**: COP 지도 위에서 전술적 배치를 검토합니다.

## 3. 상세 기능 가이드

### 3.1 상황 입력 및 시나리오

**Command & Control** 대시보드의 좌측 패널에서 수행합니다.

- **시나리오 선택**: 미리 정의된 데모 시나리오(예: "적 기계화부대 남하")를 선택하여 빠르게 테스트할 수 있습니다.
- **수동 입력 (SITREP)**: 상황보고 텍스트를 직접 입력하거나, 정형화된 필드(위협 유형, 위치 등)를 채워 넣습니다.

### 3.2 방책 추천 및 결과 분석

**"방책 추천 실행"** 버튼을 클릭하면 Agent가 분석을 시작합니다.

- **추천 결과 카드**: 점수가 높은 상위 3개 방책이 표시됩니다.
- **점수 Breakdown**: 위협 대응, 자원 효율, 지형 적합성 등 7가지 평가 요소별 점수를 그래프로 보여줍니다.
- **추천 사유**: "왜 이 방책이 추천되었는지"에 대한 AI의 자연어 설명을 제공합니다.

### 3.3 🗺️ 전술상황도 (COP) 시각화

방책 추천 결과 하단의 **"전술 상황도 (COP)"** 탭에서 작전 계획을 지도상에 시각화합니다.

#### 주요 기능
- **위협 시각화**: 적군 유닛의 위치(Red 심볼) 및 위협 반경 표시.
- **방책 시각화**: 아군 대응 부대의 배치(Blue 심볼) 및 기동 경로(화살표) 표시.
- **표준 심볼**: MIL-STD-2525D 규격의 군사 기호를 사용합니다.

#### 데이터 매핑 원리
시스템은 텍스트로 된 방책을 분석하여 다음과 같이 시각화합니다:
1.  **위치 매핑**: "철원 지역 방어" → 철원 좌표에 심볼 배치.
2.  **경로 생성**: "공격 헬기 타격" → 아군 기지에서 목표 지점까지의 비행 경로 생성.

> [!TIP]
> 지도가 검은색으로 나오거나 표시되지 않는 경우, 관리자에게 문의하여 오프라인 맵 서버 상태를 확인하세요.

## 4. 시스템 논리 이해 (심화)

이 시스템은 단순한 텍스트 매칭이 아닌, **"팔란티어 모드(Palantir Mode)"**라는 정교한 추론 엔진을 사용합니다.

### 4.1 하이브리드 추론
- **온톨로지(Ontology)**: 군사 교리와 지식 체계를 그래프로 연결하여 논리적 타당성을 검증합니다.
- **LLM (AI 모델)**: 상황의 맥락을 이해하고 창의적인 대안을 제시합니다.

### 4.2 7가지 핵심 평가 요소
방책의 점수는 다음 7가지 요소를 종합하여 산출됩니다.

1.  **위협 대응 (Threat)**: 적 위협을 얼마나 효과적으로 제거하거나 무력화하는가?
2.  **자원 가용성 (Resources)**: 작전에 필요한 자원이 현재 가용 상태인가?
3.  **전력 능력 (Assets)**: 투입되는 아군 부대의 전투력(화력, 사기 등)이 충분한가?
4.  **환경 적합성 (Environment)**: 현재의 지형 및 기상 조건에서 수행 가능한 작전인가?
5.  **효과성 (Historical)**: 유사한 과거 사례나 교리적 관점에서 효과적인 방책인가?
6.  **연계성 (Chain)**: 위협-대응-임무 간의 논리적 관계가 유기적으로 연결되는가?
7.  **임무 부합성 (Mission)**: 상급 부대에서 부여된 임무 목적과 얼마나 일치하는가?
